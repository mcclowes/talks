# Getting set up

Two tracks today — follow whichever fits:

## **Track A** — Browser only

- You need a Claude account
- Free tier is fine (but some features are premium)

## **Track B** — Hands-on with Claude Code

- You need Node.js 18+ and a terminal

---

# Create a Claude account

Go to **claude.ai** and sign up — free tier is fine.

{{qr:https://claude.ai|Sign up at claude.ai}}

---

# Get the Claude app

Available on desktop (macOS, Windows) and mobile (iOS, Android).

{{qr:https://claude.ai/download|Download Claude}}

---

# Install Claude Code (Track B)

Open your terminal and run:

```bash
npm install -g @anthropic-ai/claude-code
```

Then navigate to any project and start:

```bash
cd your-project
claude
```

You'll be prompted to authenticate with your Claude account.

{{qr:https://docs.anthropic.com/en/docs/claude-code|Claude Code docs}}
