# Documentation is still done the hard way

**Style inconsistency** — Different writers, different voices, no enforcement.

**Changelog tedium** — Every release, someone has to parse every commit.

**Stale content** — Code ships. Docs don't follow.

**No tooling feedback loop** — Docs live outside CI. Nobody catches issues early.

**Blank page paralysis** — Knowing what to write but not how to start.
