# Bad smells

- Content length
- How much contextual info are you having to provide?
- Are you having to describe in multiple ways? Different mediums?
- Should this be UI copy? Tooltips?
- Content not fit for target user persona
  - Docs for non-technical users
- Are you struggling to create screenshots, videos?
- If you can't self serve (the info), they can't self serve
